# NX POC

Minimal demonstration repository for NX.

**Goal:**
Showcase the key features of NX:
- Dependency graph
- Affected builds detection
- Computation cache

## Quick Start

1. See [nx-overview.md](docs/nx-overview.md) for a brief overview of NX and its benefits.
2. Follow the detailed walkthrough in [nx-poc-demo.md](docs/nx-poc-demo.md) for step-by-step instructions.
